# README

## How to install toolchain

### Alire

Visit the link below to install Alire

- <https://github.com/alire-project/alire/releases>

### Toolchain

```bash
$ alr toolchain --select
```

## How to create a project

```bash
$ alr init --bin <project_name>
$ cd <project_name>
$ alr build
$ alr run
```
